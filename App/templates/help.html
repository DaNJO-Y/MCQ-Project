Getting Started

Welcome to the MCQ Bank! This platform helps teachers create and manage multiple-choice questions efficiently.


  • Navigation:
  • • Access the question bank by selecting 'My Questions'.
  • • Access your created exams by selecting 'My Exams'.
  • • Import questions using the given format by clicking 'ImportQuestions'.
  • • Access this help page by clicking 'Help'.

Creating Questions

Follow these steps to add a question:

  • •Click 'New Question' at the top of any page.
  • •Enter the question text, question course code and question difficulty.
  • •Upload a main image if necessary.
  • •Upload an image (optional) and enter answer choices.
  • •More answer choices can be added when you select '+ Add Answer Options'.
  • •Mark the correct answer and save.
  • •You can view and manage your questions by selecting the 'My Questions' button.

Creating Exams

To create an exam, follow these steps:

  • •Click 'New Question' at the top of any page.
  • •Enter the Exam Title on the right, in the Exam Builder.
  • •Select questions from the question bank to be added to the exam.
  • •Use the search or filter function to find relevant questions.
  • •Questions can be unselected if no longer wanted.
  • •Select 'Shuffle answer' if you want the answers in a different order.
  • •Click and hold the drag handle (☰) next to a question. to reorder the questions.
  • •Click 'Save' to store the exam to the 'My Exams' page, where you can view and manage your exams.
  • • Select 'Download' to have the exam downloaded to your device.

Managing Exams

To manage your exams, go to the My Exams page where you can:

  • •View an exam: Select the exam title to view its details.
  • •Edit an exam: Open the exam in an editing page, similar to the new exam creation page, to make changes.
  • •Delete an exam: Delete any exam you no longer need.
  • •Download an exam: Download the exam as a PDF to your device.
  • •Search for exams: Use the search bar to find exams by title or keyword.
  • •Filter exams: Filter exams using tags or course codes to easily find relevant exams.

Managing Questions

To manage your questions, go to the My Questions page where you can:

  • •View a question's details: Select the question title to view more details, including answer options, correct answers, and usage history.
  • •Edit a question: Open the question in an editing page to make changes to the text, answer options, or tags.
  • •Delete a question: Delete any question you no longer need. You will not be allowed to delete any questions that have been used in an exam.
  • •Search for questions: Use the search bar to find questions by title, keyword, or tags.
  • •Filter questions: Filter questions by tags, course codes, or other relevant criteria to find the questions you need more easily.
